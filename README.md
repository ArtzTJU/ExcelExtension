# ExcelExtension
Extension for MS Office Excel
Uses ExcelDna 

Reads data from a csv file which should be placed in the same folder as the addon

## Provides new functions to excel
* Proteins
* Carbohydrates
* Fats
* Calories

---

#### Tech Stack:
* C#
* ExcelDNA

#### Tool Stack:
* Visual Studio 2017
* SourceTree
* Microsoft Excel 2016
